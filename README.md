# popock-login

# install dev environment
https://cli.vuejs.org/guide/prototyping.html
```
npm install -g @vue/cli @vue/cli-service-global
```

# clone
```
git clone https://github.com/scenaristeur/popock-login.git
cd popock-login
npm install
```

# launch
```
npm run serve
```
# build
```
npm run build
```
